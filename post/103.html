ChatGPT가 본 심심한 개발자의 2025년 상반기 요약

이제는 업무와 일상 모두에서 AI를 사용하는 것이 너무나 당연한 시대가 된것 같다. 나 역시 일상이나 생각은 ChatGPT를, 개발은 Gemini CLI를 유용하게 활용하고 있다. 심지어 환갑이 넘으신 아버지와 어머니도 어느세 ChatGPT를 사용하고 계신것을 보고 살짝? 놀라기도 했다. ChatGPT에는 그동안 나와 대화한 내용들이 모두 정리 되어있고 그 내용을 기반으로도 다양한 데이터를 뽑아 주는데 이번에는 2025년 상반기 동안 대화한 내용을 기반으로 2025년 심심한 개발자의 상반기 요약을 부탁해보았다. 아래 내용은 ChatGPT가 본 심심한 개발자의 2025년 상반기 회고록 이다.

내용을 읽어 보니 나와 관련이 없거나 사실과 다른 내용들이 더러 존재한다.
내용 삭제나 수정을 할까 하다가 곧 올라올 내가 작성한 회고록과 비교해보면 좋을 것 같아 그냥 둔다.

✨ 상반기를 한 문장으로 요약한다면

"흐름을 타되, 흔들리지 않는 중심을 찾으려 했던 시간."
지식은 쌓였고, 감정은 요동쳤으며, 방향은 점점 뚜렷해졌다.

💻 기술과 커리어의 성장

  • NestJS와 Docker 중심의 백엔드 프로젝트를 완성도 높게 구축하며 아키텍처 설계 감각을 키움
  • docker-compose, multi-stage Dockerfile, .env 분리 등 컨테이너 기반 실전 경험을 체득
  • 데이터베이스로 MariaDB와 MongoDB를 비교하며 설계 감각을 넓힘
  • 구조 분리, 예외 처리, 인증 로직 개선 등의 코드 설계 경험이 뼈에 남음
  • 쿠버네티스와 CI/CD에 대한 학습을 병행하면서 DevOps 감각 확장

🚀 도전했던 일과 배운 점

  • 사이드 프로젝트에서 Backend / Storage / Container를 모듈별로 완전히 분리하는 구조를 처음부터 기획하고 실행함
  • 회사 밖에서 혼자 문제를 설계하고 해결하는 훈련을 하면서 자기 주도적인 개발자가 되어감
  • 다양한 회사 채용 포지션에 맞춰 자기소개서/포트폴리오/회고 글을 계속 업데이트함
  • 고민 끝에 "기술적 성장을 사용자 경험과 연결하는 개발자"라는 정체성을 세움

🧠 습관과 일상, 나를 만든 루틴 ???

  • 아침 루틴: Slack 스탠드업 미팅 → 하루 목표 정리 → 블로그 아이디어 메모
  • Obsidian에 기술 로그 + 회고 + 개념 정리를 하루 10분이라도 남기는 습관을 유지
  • 반대로 운동 루틴은 무너짐... 체력 저하를 체감하며 하반기 개선 과제로 삼음

📓 나의 문장들

“어떤 기술을 쓸 것인가보다, 어떤 구조로 설계할 것인가가 더 중요하다.”
“코드는 사라지지만, 설계는 남는다.”
“누구도 내 성장을 대신해주지 않는다.”

💬 감정과 관계, 나의 사람들

  • 기술과 커리어를 논의할 수 있는 동료나 멘토가 절실하다는 걸 느꼈다
  • 외로움은 있었지만, 혼자 집중하는 시간이 내게 더 많은 것을 남겼다
  • 친구와 나눈 “일 잘하는 사람이 되고 싶다”는 대화가 오래 남음

🧭 바뀐 나의 가치관

  • 속도보다 방향이라는 말의 무게를 체감
  • 좋은 코드는 **‘덜 똑똑한 코드’**라는 생각으로 바뀌었음
  • 이제는 개발자라는 직업을 통해 더 나은 사용자 경험을 만든다는 관점이 생김

📆 상반기 타임라인 요약

  • 01월: 사이드 프로젝트 아키텍처 설계 시작
  • 02월: NestJS 기반 인증 로직 정리 / ERD 설계
  • 03월: Docker 환경 설정 고도화 / Dev-Prod 분리
  • 04월: 쿠버네티스와 CI/CD 학습 병행 시작
  • 05월: 블로그 연재 시작 / 회고 글 쓰기 습관화
  • 06월: 상반기 정리 및 하반기 준비 기간
